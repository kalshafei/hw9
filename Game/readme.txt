Escape the Room
You are trying to reach the green tile on the map.
Blue tiles act as switches and can be activated when moved over.
Red Lines are gates and can be manipulated by moving over blue tiles
Cyan blocks are dangerous and kill the player.

Only two maps have been implemented.
First Map: Introduces tile switches and gates.
Second Map: Introduces multiswitches and multigates as well as moving enemy blocks.

Controls:
Left - 	Arrow left 		-Move Left
Right - Arrow right		-Move Right
Up - 	Arrow Up		-Move Up
Down - 	Arrow Down		-Move Down
A - 	(Z on the keyboard)	-Continue to game from start screen
Select- (Backspace on Keyboard)	-Return to start screen (can be done at any time)
